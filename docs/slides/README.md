# 答辩材料

本目录用于存放答辩相关材料，如PPT、演示文稿等。

## PPT制作建议

### 内容结构

1. **封面**
   - 题目、姓名、学号、指导教师

2. **目录**
   - 简要说明答辩结构

3. **设计任务概述**
   - 发动机基本参数
   - 设计任务说明

4. **动力计算**
   - 数学模型简述
   - 关键计算结果展示
   - 典型曲线展示

5. **强度校核**
   - 校核部位说明
   - 计算结果
   - 安全系数结论

6. **设计说明**
   - 发动机总体结构
   - 关键零部件设计

7. **工程图纸展示**
   - 纵剖面图要点说明

8. **总结**
   - 主要结论
   - 收获与体会

9. **致谢**

### 时间控制

建议答辩时间控制在10-15分钟。
