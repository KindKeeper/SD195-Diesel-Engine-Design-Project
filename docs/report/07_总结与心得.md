# 第七章 总结与心得

## 7.1 设计工作总结

### 7.1.1 完成的主要工作

本次课程设计完成了以下工作：

1. **动力计算**
   - 建立了活塞运动学和动力学数学模型
   - 编写了完整的动力计算程序
   - 计算了0°~720°范围内各曲轴转角的运动学和动力学参数
   - 生成了附表1~5和附图1~5

2. **强度校核**
   - 对曲轴曲柄销圆角进行了安全系数计算
   - 对连杆小端进行了强度和刚度校核

3. **工程图纸**
   - 绘制了SD195柴油机纵剖面图（A0幅面，1:1比例）

### 7.1.2 设计结果分析

**运动学分析**：
- 活塞最大速度出现在曲轴转角约75°和285°附近
- 活塞最大加速度出现在上止点（0°和720°）

**动力学分析**：
- 最大气体力出现在爆发压力峰值附近（约370°）
- 最大往复惯性力出现在进气上止点（0°）
- 侧向力在活塞上下运动时方向交替变化

**强度校核结果**：
- 曲轴安全系数满足设计要求
- 连杆小端强度和刚度满足设计要求
