# SD195柴油机设计项目

山东大学内燃机研究所《内燃机设计》课程设计项目。

## 项目概述

本项目为SD195单缸农用卧式水冷四冲程柴油机的设计与分析，包括动力计算、强度校核和工程制图。

### 发动机主要参数

| 参数 | 数值 |
|------|------|
| 额定功率 | 8.8 kW |
| 额定转速 | 2000 r/min |
| 气缸直径 | 95 mm |
| 活塞行程 | 115 mm |
| 连杆比 | 0.329 |

## 使用说明

### 动力计算（全流程自动化）

本项目实现**全流程自动化**：运行一次程序，自动生成所有计算结果和曲线图。

#### 前置要求

**安装Miniforge（Python环境管理工具）**
1. 访问 https://github.com/conda-forge/miniforge/releases
2. 下载 `Miniforge3-Windows-x86_64.exe`（约100MB）
3. 双击安装，**勾选"Add to PATH"**



#### 快速开始（4步完成全部计算）

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 安装Miniforge | 下载并安装Miniforge（只需一次） |
| 2 | 双击 `安装环境.bat` | 创建Python环境，安装依赖（只需一次） |
| 3 | 双击 `运行计算.bat` | **一键完成**：计算 → 生成附表 → 生成附图 |
| 4 | 查看结果 | 所有成果已就绪，可直接插入设计说明书 |

#### 自动生成的成果

运行 `运行计算.bat` 后，程序会自动生成以下文件到 `results/` 目录：

```
results/
├── tables/              # 📊 附表1-5（CSV格式）
│   ├── 附表1_活塞运动学数据.csv
│   ├── 附表2_气体力与合力.csv
│   ├── 附表3_连杆及侧压力.csv
│   ├── 附表4_连杆轴颈负荷.csv
│   └── 附表5_主轴颈负荷.csv
│
├── calculations/        # 📝 强度校核报告
│   └── 强度校核报告.txt
│
└── plots/               # 📈 附图1-5（PNG格式）
    ├── 附图1_位移速度加速度曲线.png
    ├── 附图2_气体力_惯性力_合力曲线.png
    ├── 附图3_侧压力_连杆力_切向_径向力曲线.png
    ├── 附图4_连杆轴颈负荷极坐标图.png
    └── 附图5_主轴颈负荷极坐标图.png
```

**说明**：
- `results/` 目录下的内容由程序**自动生成**，已添加到 `.gitignore`
- CSV文件可用Excel/WPS直接打开、编辑和打印
- 如需备份成果，可复制到其他位置或从GitHub Release下载

详细说明见 `Python环境使用说明.md`。
