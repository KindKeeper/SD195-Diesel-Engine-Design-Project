# 计算结果数据表

本目录存放动力计算程序生成的计算结果数据表（CSV格式）。

## ⚠️ 重要提示

**本目录下的CSV文件由程序自动生成，不应手动修改或提交到Git仓库。**

如需保留成果，请在运行程序后备份到其他位置。

## 文件说明

运行程序后，本目录将包含以下文件：

| 文件名 | 对应附表 | 内容说明 |
|--------|---------|---------|
| `附表1_活塞运动学数据.csv` | 附表1 | 活塞位移、速度、加速度（0°~360°） |
| `附表2_气体力与合力.csv` | 附表2 | 气体力、惯性力、合力（0°~720°） |
| `附表3_连杆及侧压力.csv` | 附表3 | 连杆力、侧压力、切向力、径向力（0°~720°） |
| `附表4_连杆轴颈负荷.csv` | 附表4 | 连杆轴颈负荷（0°~720°） |
| `附表5_主轴颈负荷.csv` | 附表5 | 主轴颈负荷（0°~720°） |

## 使用方式

**CSV文件可用Excel或WPS直接打开、编辑和打印**：

1. **查看数据**：用Excel双击打开CSV文件
2. **调整格式**：调整列宽、字体、边框等
3. **打印附表**：直接打印或复制到Word文档
4. **插入文档**：复制表格内容，粘贴到设计说明书

## 格式说明

- **文件格式**：CSV（逗号分隔值）
- **编码方式**：UTF-8（支持中文）
- **数据单位**：
  - 力：kN（千牛）
  - 位移：mm（毫米）
  - 速度：m/s（米/秒）
  - 加速度：m/s²（米/秒²）
  - 角度：度（°）
