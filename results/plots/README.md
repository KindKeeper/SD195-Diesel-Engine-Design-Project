如需保留成果，请在运行程序后备份到其他位置。

## 使用方式

**PNG图片可直接插入Word文档**：

1. 在Word中选择"插入" → "图片"
2. 选择本目录中的PNG文件
3. 调整图片大小和位置

## 图表用途

这些图表用于插入设计说明书附录，作为动力计算结果的可视化展示。
