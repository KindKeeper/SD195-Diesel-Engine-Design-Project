# 图纸导出文件

本目录存放CAD图纸导出的PDF文件，用于打印和提交。

## 文件说明

- `SD195_纵剖面_A0_1比1.pdf` - 纵剖面图PDF
  - 幅面：A0
  - 比例：1:1
  - 用于提交和打印

- `SD195_零件图_若干.pdf` - 零件图PDF（如有）

## 导出设置

### 从AutoCAD导出PDF

1. 命令行输入 `EXPORTPDF` 或 `PUBLISH`
2. 选择输出范围：窗口/布局
3. 图纸尺寸：A0
4. 输出质量：矢量质量最高
5. 线宽：按图层或按颜色

### 从FreeCAD导出PDF

1. 进入 TechDraw 工作台
2. 创建图纸页面，选择A0模板
3. 插入视图
4. 文件 → 导出 → PDF

## 打印建议

- 使用A0幅面打印机或到打印店打印
- 选择白色绘图纸
- 线宽设置确保清晰可读
