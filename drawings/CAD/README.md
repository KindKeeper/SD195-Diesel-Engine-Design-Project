# CAD 源文件

本目录存放工程图纸的CAD源文件。

## 文件说明

- `SD195_纵剖面.dwg` 或 `SD195_纵剖面.FCStd` - 纵剖面图CAD源文件
  - 幅面：A0
  - 比例：1:1
  - 活塞位置：上止点
  - 气缸中心线：居中

- `SD195_典型零件图.dwg` - 典型零件图（如曲轴、连杆）
  - 用于辅助理解发动机结构

## 软件建议

- **AutoCAD**: 使用 `.dwg` 格式
- **FreeCAD**: 使用 `.FCStd` 格式（开源免费）

## 制图要求

1. **幅面和比例**
   - A0幅面（841mm × 1189mm）
   - 比例 1:1

2. **视图布局**
   - 活塞位于上止点位置
   - 气缸中心线居中
   - 包含必要的局部放大图

3. **标注要求**
   - 主要尺寸标注
   - 配合尺寸标注
   - 形位公差标注
   - 表面粗糙度标注

4. **技术要求**
   - 在图纸空白处列出技术要求
   - 材料标注
   - 热处理要求

5. **标题栏**
   - 按国家标准填写标题栏
   - 包含：图名、图号、材料、比例、设计者、日期等
